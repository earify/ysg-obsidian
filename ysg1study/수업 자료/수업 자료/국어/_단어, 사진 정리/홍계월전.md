### 특징 :
- 갈래 : 고전 소설
- 성격 : **일대기적**, 영웅적, 전기적
- 구성 : 7회의 회장체 구성, 영웅의 일대기적 구성
- 제재 : **홍계월의 영웅성**
- 주제 : 여성인 홍계월의 영웅적 활약상
- 특징 : **여성 > 남성**, 남성이 나약함
- 의의 : 여성의 우월함 -> **기존 소설에서 벗어남**
